# Vidos

**Enterprise verification infrastructure for digital identities and verifiable credentials.**

Vidos provides the verification layer that enables organisations to accept digital credentials from EU Digital Identity Wallets, UK DIATF-certified wallets, and other standards-compliant identity systems through a single integration.

## What We Build

| Service | Description |
|---|---|
| **[Authorizer](https://vidos.id/products/vidos-authorizer)** | Request and receive verified credentials from user wallets via OpenID4VP |
| **[Gateway](https://vidos.id/products/vidos-gateway)** | Access all services through one unified API endpoint |
| **[Resolver](https://vidos.id/products/vidos-universal-resolver)** | Resolve DIDs across all major methods through a single API |
| **[Validator](https://vidos.id/products/vidos-validator)** | Validate credentials against your business rules and policies, no code changes needed |
| **[Verifier](https://vidos.id/products/vidos-verifier)** | Verify credential authenticity with cryptographic proof across all major formats |

## Standards and Compatibility

W3C Verifiable Credentials &middot; OpenID4VP / OpenID4VC &middot; ISO 18013-5 (mDL) &middot; SD-JWT &middot; JSON-LD &middot; JWT &middot; mDOC &middot; vLEI &middot; DCQL

## Regulatory Compliance

- **eIDAS 2.0** &mdash; Verification infrastructure for EUDI Wallet acceptance across all 27 EU Member States
- **UK DIATF** &mdash; Certified as Orchestration, Component, and Attribute Service Provider on the [DVS Register](https://www.digital-identity-services-register.service.gov.uk/provider-details?providerId=63)
- **ISO 27001:2022** &mdash; Certified information security management
- **ISO 9001:2015** &mdash; Certified quality management
- **UK Cyber Essentials** &mdash; Government-backed cyber security certification
